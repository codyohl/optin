# Toll-Free Number Verification Opt-In

A simple, professional webpage for collecting toll-free number verification opt-in requests.

## Features

- Clean, modern responsive design
- Form validation for toll-free numbers (800, 888, 877, 866, 855, 844, 833 series)
- Client-side validation with visual feedback
- Success message confirmation
- Mobile-friendly layout

## Usage

This webpage is designed to be hosted on GitHub Pages. Users can submit their information to opt-in for toll-free number verification.

### Form Fields

- **Full Name** (required)
- **Toll-Free Number** (required) - Validates proper toll-free number format
- **Email Address** (required)
- **Company Name** (optional)
- **Consent** (required) - Agreement to comply with telecommunications regulations

## Deployment

To deploy this webpage on GitHub Pages:

1. Go to your repository settings
2. Navigate to "Pages" section
3. Select the branch you want to deploy from
4. The site will be available at: `https://codyohl.github.io/optin/`

## Technical Details

- Pure HTML/CSS/JavaScript (no dependencies)
- No backend required for the static page
- Ready to integrate with a backend API for data collection
- Accessible and semantic HTML
